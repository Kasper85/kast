# Screenshots

Placeholder para capturas de pantalla de la app.

Para agregar screenshots:

1. Tomar capturas de la app en un dispositivo o emulador
2. Guardar como:
   - `home.png` — pantalla principal
   - `movie-detail.png` — detalle de película
   - `tv-settings.png` — configuración de TV
3. Colocar en esta carpeta
4. Actualizar las referencias en README.md

Las capturas deben ser:
- Resolución mínima: 1080x1920 (portrait)
- Formato: PNG
- Sin información sensible (tokens, IPs, datos personales)
