# Comparativas — Kast vs Alternativas

Comparación objetiva de Kast con las principales alternativas para controlar TVs LG webOS desde un celular Android.

---

## Kast vs Web Video Caster

| Característica | Kast | Web Video Caster |
|---------------|------|------------------|
| Precio | Gratis (MIT) | Freemium |
| Películas | ✅ TMDB | ❌ No |
| Series | ✅ Con temporadas/episodios | ❌ No |
| Favoritos | ✅ Locales (Room) | ❌ No |
| Historial | ✅ Local (Room) | ❌ No |
| TV LG webOS | ✅ SSAP nativo | ✅ DLNA/UPnP |
| Discovery de TV | ✅ SSDP automático | ❌ Manual |
| Chromecast | ❌ No | ✅ Sí |
| Open source | ✅ MIT | ❌ No |
| Interface moderna | ✅ Material 3 | ⚠️ Básica |

**Resumen:** Kast es mejor para discover de películas y envío directo a LG. Web Video Caster es más versátil para Chromecast y otros protocolos.

---

## Kast vs LG ThinQ

| Característica | Kast | LG ThinQ |
|---------------|------|----------|
| Precio | Gratis | Gratis |
| Películas | ✅ TMDB | ❌ No |
| Control TV | ✅ SSAP nativo | ✅ Nativo LG |
| Discovery | ✅ SSDP | ✅ Integrado |
| Favoritos | ✅ Películas | ❌ No |
| Requiere cuenta LG | ❌ No | ✅ Sí |
| Open source | ✅ MIT | ❌ No |
| Funciona sin cuenta | ✅ Sí | ❌ No |

**Resumen:** Kast no requiere cuenta LG y agrega TMDB + favoritos. ThinQ es más completo para funciones nativas de la TV.

---

## Kast vs Kodi

| Característica | Kast | Kodi |
|---------------|------|------|
| Precio | Gratis | Gratis |
| Plataforma | Android | Multiplataforma |
| Películas | ✅ TMDB | ✅ Add-ons |
| Series | ✅ Con temporadas/episodios | ✅ Add-ons |
| Favoritos | ✅ Locales | ✅ Locales |
| Envío a TV LG | ✅ SSAP nativo | ❌ No directo |
| Complejidad | ✅ Baja | ⚠️ Media-Alta |
| Personalización | ✅ Limitada | ✅ Extensa |
| Streaming propio | ❌ No | ✅ Sí (con add-ons) |

**Resumen:** Kodi es más potente y personalizable, pero Kast es más simple y está diseñado específicamente para el flujo de enviar a TV LG.

---

## Kast vs Plex

| Característica | Kast | Plex |
|---------------|------|------|
| Precio | Gratis | Freemium |
| Películas | ✅ TMDB | ✅ TMDB + propio |
| Series | ✅ Con temporadas/episodios | ✅ Con temporadas/episodios |
| Servidor | ❌ No | ✅ Requiere |
| Favoritos | ✅ Locales | ✅ En servidor |
| Envío a TV LG | ✅ SSAP nativo | ❌ DLNA/Chromecast |
| Open source | ✅ MIT | ❌ No |
| Almacenamiento | Solo celular | Requiere disco |

**Resumen:** Plex es un sistema completo de media server. Kast es una app ligera para discover + envío a TV LG.

---

## Kast vs Jellyfin

| Característica | Kast | Jellyfin |
|---------------|------|----------|
| Precio | Gratis | Gratis |
| Open source | ✅ MIT | ✅ GPL |
| Películas | ✅ TMDB | ✅ TMDB + propio |
| Servidor | ❌ No | ✅ Requiere |
| Favoritos | ✅ Locales | ✅ En servidor |
| Envío a TV LG | ✅ SSAP nativo | ⚠️ DLNA |
| Complejidad | ✅ Baja | ⚠️ Media |
| Multiusuario | ❌ No | ✅ Sí |

**Resumen:** Jellyfin es un media server completo. Kast es una app ligera sin servidor.

---

## Resumen

| Si necesitas... | Usa... |
|----------------|--------|
| Solo enviar películas a TV LG | **Kast** |
| Control completo de media server | **Plex** o **Jellyfin** |
| Personalización extrema | **Kodi** |
| Chromecast + múltiples protocolos | **Web Video Caster** |
| Funciones nativas de la TV | **LG ThinQ** |
| App simple y rápida para LG | **Kast** |

> Kast no pretende reemplazar a estas alternativas. Es una herramienta específica para el flujo de buscar → ver → enviar a TV LG.
